Grande variedade de edições
» Web Edition

Uma poderosa plataforma de aplicações e serviços web concebidos exclusivamente como um servidor voltado para a Internet. Ela oferece uma administração melhorada e ferramentas de diagnóstico para ajudar a reduzir os custos de infra-estrutura.

» Standard Edition

O mais robusto sistema operacional Windows Server até o momento. Ele oferece ferramentas poderosas que dão a você mais controle sobre seus servidores, agiliza e simplifica a configuração de tarefas de gestão.

» Enterprise Edition

Uma plataforma de servidor avançada que fornece uma solução de custo eficaz e confiável para projetos críticos. Ele oferece recursos inovadores para economia de energia e gerenciamento do servidor.

» Datacenter Edition

Uma plataforma de classe empresarial para a implantação de aplicações críticas de negócio em servidores pequenos e grandes. Ele melhora a disponibilidade e melhora o gerenciamento de energia.

Ferramentas de banco de dados
Poderoso banco de dados, o Servidor SQL 2008 R2 oferece recursos inovadores que irão permitir que sua empresa expanda com confiança suas operações de bancos de dados, melhore a eficiência de TI e desenvolvimento, e possibilite Business Intelligence altamente expansível e bem gerenciado de maneira self-service para seus usuários.

  • Plataforma confiável e escalável
  • Mais eficiência para desenvolvedores
  • Baixo custo, mais benefícios
Plataforma Web da Microsoft

  •   Framework ASP.NET

    Um completo framework que ajuda você a criar soluções de ponta para web facilmente.

  •   IIS Web Server

    Hopede com confiança aplicações em ASP.NET e também aplicações web em PHP.

  •   SQL Server

    Um poderoso motor de banco de dados completo para executar e criar poderosos bancos dados.

  •   Developer Tools

    Ferramentas (Built-in) de desenvolvimento para construção de sites, aplicações e serviços baseados em padrões web.

Instalador fácil de usar
O Microsoft Web Platform Installer é uma ferramenta gratuita que facilita para obter as últimas atualizações de componentes da plataforma da Microsoft. Além disso, facilita a utilização dos mais populares aplicativos web gratuitos para blogs, gerenciamento de conteúdo e muito mais.

  • Disponível para download gratuito
  • Sempre inclui as atualizações mais recentes
  • Aplicativos web populares podem ser instalados facilmente
