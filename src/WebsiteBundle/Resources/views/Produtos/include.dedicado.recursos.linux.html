CentOS, Ubuntu ou Debian

Seu servidor será pré-configurado e ativo com a sua distribuição Linux favorita. CentOS, Ubuntu ou Debian, é da sua escolha!

Acesso Shell (SSH) completo

Você será o administrador do seu servidor. Desfrute do acesso root completo para instalar qualquer software que você precisa.

O ideal LAMP (servidor web pronto)

Linux, Apache, MySQL e PHP / Perl ou Python (LAMP) utilizado por milhões de web sites e aplicações em todo o mundo. Transforme seu novo Linux Server em uma poderosa plataforma LAMP usando o apt-get ou yum.

Plataformas Web Open Source

Um servidor executando o LAMP suporta uma ampla gama de websites prontos e plataformas de gerenciamento de conteúdo como Wordpress, Drupal, Joomla, bem como centenas de scripts gratuitos de loja virtual, wiki e galeria de fotos. Basta instalá-lo e começar a usar!

Enorme comunidade de desenvolvedores

A popularidade do Linux e sua aproximação com a comunidade torna fácil de encontrar ajuda e recursos online. Quem sabe, uma vez que estiver a bordo, você possa ficar viciado em seu servidor e talvez você acabe contribuindo código para a próxima versão de sua distribuição Linux favorita!
